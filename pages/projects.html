[ PACT.AR ]

     Production SaaS platform, self-provisioned and managed on Oracle Cloud.
     A simple and legally valid way to create and sign digital contracts, valid
under Argentine law 25.506.
     All in a single platform, it integrates with Mercado Pago for subscription
billing, making the entire process paperless, fast, and accessible, specially
useful for freelancers and small businesses.

     Stack involved:
     - Frontend: Next.js/React, Tailwind.
     - Backend: Node.js/Express, PostgreSQL.
     - Deployment: Coolify + GitHub Actions hosted on Oracle Cloud.
